## 🛡️ Security Posture & Compliance
This infrastructure is designed to align with **NIST 800-53** and **CIS Benchmarks**. Key security features include:
* **Micro-segmentation:** Zero-trust communication between Production and Management zones.
* **Immutable Backups:** Ransomware-proof storage using AWS S3 Object Lock.
* **Automated Response:** Wazuh SIEM automatically triggers n8n workflows to block malicious IPs.
