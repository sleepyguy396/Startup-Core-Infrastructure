# Startup-Core-Infrastructure
End-to-end IT/Security infrastructure for a scaling startup, featuring vSphere, Cloud-Hybrid architecture, and n8n automation.

# 🏢 Startup Enterprise Infrastructure (Project: Zero-to-Scale)
This repository documents the end-to-end design, implementation, and security of a high-growth startup's IT infrastructure.

## 🛠️ The Stack
- **Virtualization:** VMware vSphere (ESXi/vCenter)
- **Networking:** pfSense Firewall / VLAN Segmentation
- **Automation:** n8n / Python / PowerShell
- **Identity:** Windows Server Active Directory (IAM)
